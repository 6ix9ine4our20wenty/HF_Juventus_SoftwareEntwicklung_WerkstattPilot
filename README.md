Arbeitsgruppe 3 / Livio Luna / Dominic Lolos

Import in Eclipse:
1. File > Import > Existing Projects into Workspace
2. Ordner werkstattPilot auswählen
3. Projekt importieren und JDK 21 verwenden

Starten:
- Main-Klasse: ch.werkstattteam.werkstattpilot.presentation.WerkstattPilotApp

Tests:
- ch.werkstattteam.werkstattpilot.test.persistence.AuftragRepositoryTest
- ch.werkstattteam.werkstattpilot.test.business.AuftragServiceTest

Hinweis:
Der Prototyp verwendet eine Mock-DB im Hauptspeicher und harte Testdaten. Es werden keine Benutzereingaben und keine echte Datenbank benötigt.
Die Javadoc-Dokumentation liegt im Ordner javadoc_html. Einstieg: javadoc_html/index.html
